# Boston Barbell — Content strategy (Phase 1)

Internal HTML presentation. Open `index.html` in a browser, or view the **GitHub Pages** URL after you enable hosting (see below).

## Live site (GitHub Pages)

**https://samsuthrland.github.io/boston-barbell-content-strategy/**

GitHub Pages is enabled on the `main` branch (root). If the page shows 404, wait a minute and refresh; check **Settings → Pages** for the build status.

## Local preview

```bash
open index.html
```

Or run a local server: `python3 -m http.server 8080` then open `http://127.0.0.1:8080/`.

- Strategy source PDFs — reference documents (optional in repo; remove if the repo should stay presentation-only).
